# AI Business Question & Data Quality Sprint

## Project Overview

This project focuses on understanding and inspecting a real-world Netflix Movies and TV Shows dataset before performing deep analysis.

The main objective of the project is to:

* understand the dataset structure,
* identify business-focused questions,
* inspect data quality issues,
* and create a cleanup and analysis plan using both analytical thinking and AI support.

---

# Dataset Used

**Dataset Name:** Netflix Movies and TV Shows Dataset

The dataset contains information about Netflix content such as:

* Movies and TV Shows
* Directors
* Cast members
* Country
* Release year
* Ratings
* Duration
* Genres
* Descriptions

# Business Questions

* Which genre contains the highest number of Movies and TV Shows on Netflix?
* Which countries produce the most Netflix content?
* What type of content (Movies or TV Shows) is added more frequently in recent years?

# Data Quality Checks Performed

* Missing value inspection
* Duplicate record checking
* Unique value inspection
* Incorrect formatting detection
* Release year inspection

# Tools & Technologies Used

# How AI Was Used

AI was used to:

* organize the project structure,
* suggest business questions,
* identify possible data quality issues,
* and generate cleanup planning ideas.

Final decisions and observations were manually reviewed before inclusion in the project.

# Conclusion

This project demonstrates the importance of understanding and cleaning data before performing deeper business analysis. Proper data quality inspection helps ensure accurate insights and better decision-making in real-world analytics projects.
